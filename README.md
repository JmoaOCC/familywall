# FamilyFlow

Aplicación web básica inspirada en FamilyWall, pensada para tablet con funciones de calendario, menú semanal y lista de la compra.

## Archivos

- `index.html`: interfaz principal y estructura de la app.
- `styles.css`: diseño tipo tablet, botones grandes, y estética limpia.
- `app.js`: lógica para manejar eventos, menús y lista de la compra con almacenamiento local.

## Cómo usar

1. Abre `index.html` en el navegador.
2. Usa los botones grandes para navegar entre Inicio, Calendario, Menú y Compra.
3. Añade eventos, platos y artículos. Los datos se guardan en `localStorage`.
